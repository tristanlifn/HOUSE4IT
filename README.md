# HOUSE4IT opgave

## How to run
1. Open in Visual studio
2. restore nuget packages (```Tools > NuGet Package Manager > Restore```)
3. Run with ```ctrl + F5```

## Use
Add each file you want to edit to the files list in main (format: ```["file.csv", "file2.csv"]``` )
Make sure all the files you want to edit is in the same directory as teh executable.
The output will be placed in your downloads folder
